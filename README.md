# Repositório criado para o Trabalho de Arquitetura de Computadores 2026/1

 Projeto desenvolvido para analisar como diferentes acessos de memória influenciam o desempenho da hierarquia da cache.

## Objetivo
Investigar experimentalmente como o impacto das diferentes formas de acesso a memória influenciam na cache.

Experimentos:
- Acesso sequencial vs acesso aleátorio
- Influência da ordem dos índices em matrizes i/j
- Multiplicação de matrizes
- Comportamento da cache em diferentes algoritmos.
